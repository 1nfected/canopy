Examples

Below are examples of all the basic commands in the framework.

Starting a Browser

start firefox
start ie
start chrome

Describe the Test

describe "some functionality should work"

Browse to a Url

url "http://lefthandedgoat.github.com/canopy/"

Assert on a Url

on "http://lefthandedgoat.github.com/canopy/"

Clicking an Element

Note: Identify the element you want to interact with by using CSS Selectors 

click "#submit"

Setting a TextBox

"#firstName" << "John"

Clear a TextBox

clear "#lastName"

Assert the Value of an Element

"#firstName" == "John"

Assert the Value of an Element Not Equal

"#firstName" != "Johnny"

Assert One of Many Elements Has a Value

".items" *= "Item 1"

Assert No Elements Have Value

".items" !*= "Item 100"

Assert Element Selected

selected "#radio1"

Assert Element Not Selected

deselected "#checkbox"

Reload Current Page

reload ()

Simulate Key Press

press down
press enter
press tab
